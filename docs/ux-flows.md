# UX Flows

## First-time user

1. Landing page
2. Register or sign in
3. Default lists are created automatically
4. User lands on the main list

## Add items

1. Open categories
2. Select products from emoji grid
3. Add to list

## Search and quick add

1. Enter search term
2. Select suggestion
3. Item added to current list

## Shopping mode

1. Open list in store
2. Toggle purchased state
3. Optional notes per item

## Favorites

1. Open favorites
2. Tap to add to list
3. Remove from favorites if needed

## Import from text

1. Open import modal
2. Paste list text
3. AI maps products and adds items

## Admin: add product

1. Open admin panel
2. Create or edit a product
3. Optionally generate/upload emoji
